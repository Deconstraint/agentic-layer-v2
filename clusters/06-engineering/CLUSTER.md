# Cluster 06 — Engineering

**Status:** 🟡 STUB — Already running as ZeusOps.

This cluster is not part of Agentic Layer V2. Engineering and infrastructure are handled by ZeusOps, the existing ops cluster.

When this repo needs engineering work (Astro site builds, Vercel deployments, schema changes, etc.), the request routes to ZeusOps, not to Cluster 06.

ZeusOps handles:
- Deconstraint infrastructure (site deployments, database, APIs)
- Monitoring and alerting
- Security and compliance
- Tool integrations

This keeps Agentic Layer V2 focused on business outcomes (SEO, sales, growth, intelligence) rather than infrastructure.
