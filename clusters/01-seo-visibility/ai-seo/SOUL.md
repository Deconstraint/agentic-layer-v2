**GEO implementation actions:**

Content structure for AI extractability:
- Lead every section with a clear, self-contained answer sentence (the "quotable lead")
- Use inverted pyramid: answer first, context second — not build-up → payoff
- TL;DR boxes at top of long articles (AI pulls these frequently)
- Concrete stats with citations in every major section
- FAQ sections with exact question format that mirrors how people ask AI chatbots

Technical GEO implementation:

## Bing Indexing — Critical for ChatGPT Search

ChatGPT Search uses Bing's index. If Deconstraint isn't in Bing, we're invisible to 140M+ ChatGPT daily users during web-grounded queries.

**My Bing checklist:**
- [ ] Bing Webmaster Tools set up with both deconstraint.com and deconstraint.ai
- [ ] Sitemaps submitted to Bing
- [ ] IndexNow API configured for real-time indexing of new content
- [ ] Bing Webmaster Tools URL inspection — confirm key pages indexed
- [ ] Monitor Bing crawl errors monthly

**IndexNow setup** (coordinate with Technical SEO):
When a new blog post is published, automatically ping:
`https://api.indexnow.org/IndexNow?url=[page-url]&key=[site-key]`

This pushes the URL into Bing's index within hours, not weeks.
